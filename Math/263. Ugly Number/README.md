**263. Ugly Number**

A **ugly number** is a number whose prime factors only include `2`, `3`, and `5`.

Given an integer `n`, return `true` if it is an **ugly number**.

**Example 1:**

```
Input: n = 6
Output: true
Explanation: 6 = 2 × 3
```

**Example 2:**

``` 
Input: n = 8
Output: true    
Explanation: 8 = 2 × 2 × 2
```
